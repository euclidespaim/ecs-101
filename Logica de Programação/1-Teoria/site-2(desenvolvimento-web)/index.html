📖 Guia de Teoria Interativa

Desenvolvimento web é sobre organizar elementos visuais e dar a eles o estilo desejado. Cada site na internet é feito de tags estruturais e regras de folhas de estilos aplicadas a essas tags.

1. O que é HTML? (Estrutura)

O HTML é o esqueleto do site. Ele define os textos, imagens, links e formulários que aparecem. Usamos tags com abreviaturas em inglês como <p> (paragraph) ou <a> (anchor/link) para demarcar os blocos.

<h1>Título da Página</h1> <p>Este é um parágrafo que contém informações úteis.</p> <a href="https://site.com">Ir para o Site</a>

Analogia: HTML é como o esqueleto do corpo humano ou os tijolos de cimento de um edifício.

2. O que é CSS? (Estilos)

Sem o CSS, todas as páginas seriam apenas textos pretos em fundo branco. Com ele, você aplica as regras visuais. Os estilos são compostos de propriedades (ex: color) seguidas de dois pontos e os valores (ex: blue).

/* Aplicando no CSS */ h1 { color: red; /* Cor do texto vermelha */ background-color: gold; /* Fundo amarelo ouro */ text-align: center; /* Alinhamento central */ }

Analogia: CSS é o acabamento: escolher a cor da tinta, papel de parede e pendurar quadros nas paredes.

3. Seletores CSS (Alvos)

Seletores informam ao CSS qual elemento ele deve colorir:

  • Tag: Escreva apenas o nome da tag (ex: h2). Altera todos os elementos idênticos.
  • Classe (.): Usada quando queremos marcar vários elementos que compartilham o mesmo estilo (ex: .importante).
  • ID (#): Usado para estilizar um único elemento exclusivo na página (ex: #topo).

/* Selecionando no CSS */ p { color: gray; } /* Altera todos os <p> */ .destaque { font-weight: bold; } /* Altera class="destaque" */ #rodape { color: silver; } /* Altera id="rodape" */

4. O Box Model (Espaçamento)

Todos os elementos na tela são caixas empilhadas. Controlar o tamanho dessa caixa e a distância até os vizinhos é fundamental:

  • Content: Onde fica o texto ou a imagem.
  • Padding: Espaço interno para respirar (afasta o texto da borda).
  • Border: A linha que desenha o limite físico da caixa.
  • Margin: Espaço externo (empurra outras caixas para longe).
Analogia: Enviar uma caneca pelos Correios: a caneca é o Content, o plástico bolha protetor é o Padding, a caixa de papelão é a Border, e o espaço até o próximo pacote no caminhão é a Margin.
⚠️ Regra de Ouro do HTML: Tags Fechadas!
Não se esqueça de fechar cada tag aberta (ex: <p>conteúdo</p>). Tags abertas incorretamente bagunçam o alinhamento da página toda!
