📖 Guia de Teoria Interativa

Imagine que você está programando um jogo de corrida. Se o carro passar pela linha de chegada, o jogo deve parar e mostrar 'Você venceu!'. Caso contrário, o carro continua correndo. Essa tomada de decisão no código é feita usando estruturas condicionais.

1. O comando 'if' (Se)

O if é o ponto de partida. Ele testa uma condição. Se essa condição for verdadeira (True), o bloco de código que está recuado (indentado) logo abaixo dele é executado.

energia = 80 if energia > 50: print("O personagem pode correr!")

Analogia: Se estiver chovendo, eu abro o guarda-chuva.

2. O comando 'else' (Senão)

O else serve para quando a condição do if for falsa (False). Ele é o plano B: se a primeira condição não deu certo, o código entra obrigatoriamente no else.

temperatura = 15 if temperatura > 25: print("Está calor.") else: print("Está frio ou fresco.")

Analogia: Se fizer sol, vou à praia. Senão, fico em casa jogando videogame.

3. O comando 'elif' (Senão Se)

E se você tiver mais de duas opções? O elif (abreviação de else if) permite testar condições extras em ordem. O Python vai testando uma por uma até achar uma verdadeira. Se nenhuma for verdadeira, ele executa o else no final.

nota = 8.5 if nota >= 9.0: print("Excelente!") elif nota >= 7.0: print("Muito bom!") else: print("Precisa estudar mais!")

Analogia: Se eu tiver R$ 20, compro pizza. Senão, se eu tiver R$ 10, compro um pastel. Senão, como pão com queijo.

4. Valores Booleanos e Operadores Lógicos

Além de comparar números, as condições avaliam em valores booleanos (lógicos): True (Verdadeiro) ou False (Falso). Podemos combinar múltiplas condições no mesmo if usando operadores lógicos:

  • and (E): Exige que todas as condições sejam verdadeiras.
  • or (OU): Exige que pelo menos uma condição seja verdadeira.
  • not (NÃO): Inverte o valor lógico (ex: not True vira False).

# Exemplo de condição composta tem_ingresso = True idade = 17 if tem_ingresso and idade >= 16: print("Acesso liberado!")

Analogia: Vou à praia se fizer sol E eu tiver dinheiro. Vou de ônibus se estiver chovendo OU se eu estiver cansado.
⚠️ Regra de Ouro do Python: Indentação!
No Python, os blocos de código pertencentes ao if, elif e else devem estar afastados da margem esquerda (geralmente por 4 espaços ou 1 tab). Isso mostra ao Python o que está "dentro" da condição.

💡 O que observar?
Perceba que quando a nota é maior que 9.0, a condição do if é satisfeita, então ele ignora as condições do elif e do else inteiramente. Se a nota for menor que 7.0, ele avalia as duas condições falsas e entra obrigatoriamente no else.
nota = 7.5
if nota >= 9.0:
print("Excelente!")
elif nota >= 7.0:
print("Muito bom!")
else:
print("Precisa estudar mais!")
